# Feature Modules

This folder keeps domain workflows isolated by module.

- `auth`: authentication, sessions, and access policies.
- `leads`: lead intake, scoring, and pipeline orchestration.
- `clients`: client entities, vaults, and account views.
- `scope-manifests`: scope-lock and discovery documentation.
- `proposals`: proposal generation, status flow, and exports.
- `pricing`: dynamic pricing components and margin controls.
- `change-orders`: out-of-scope request handling after Closed Won.
